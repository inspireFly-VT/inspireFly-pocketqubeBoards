Text file to keep track of LOCKING file when actively working on it, see How to work on KiCAD Files with GitHub on teams.

Log Template:

FILE: boardName.kicad_pcb , schematicName.kicad_sch
LOCKED BY: FULL NAME (PID)
START: DATE and TIME
STATUS: List what you are doing

Remember to remove your 'LOCK' when done and ensure that push has relevant comments to actions done.

Log:
FILE: boardName.kicad_pcb , mainboard.kicad_sch
LOCKED BY: Yoshita Papasani (PID)
START: 2:18 PM 10/9/2025
STATUS: working on IMU
